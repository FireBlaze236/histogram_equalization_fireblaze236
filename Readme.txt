Prerequisites:
===========================
1. Python
2. Numpy
3. PIL(Python Image Library) Or, Pillow
4. MatPlotLib

Usage:
===========================

command line in the directory:

python histeq.py <image_name>.<format>

if no image is provided, 
the program will automatically run with test.bmp image already in the repository.
